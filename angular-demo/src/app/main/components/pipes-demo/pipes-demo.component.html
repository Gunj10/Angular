Pipes


Pipes are simple functions to use in template expressions to accept an input value and return a transformed value.
Pipes are useful because you can use them throughout your application, while only declaring each pipe once,
Use pipes to transform strings, currency amounts, dates, and other data for display.

Date Pipes

Original date : {{ todaysdate }}
Original date using pipe : {{ todaysdate | date }}
Original date using parameterized pipe short : {{ todaysdate | date: 'short' }}
Original date using parameterized pipe medium : {{ todaysdate | date: 'medium' }}
Original date using parameterized pipe long : {{ todaysdate | date: 'long' }}
Original date using parameterized pipe full : {{ todaysdate | date: 'full' }}

Currency Pipes

Default is US dollar {{ 5 | currency }}
When used as EUR {{ 5 | currency: 'EUR' }}
When used as INR {{ 5 | currency: 'INR' }}

Uppercase and Lowercase Pipes

When used as uppercase: {{ 'Text is Uppercase' | uppercase }}
When used as lowercase: {{ 'Text is Lowercase' | lowercase }}

Decimal Pipes

No specified formatting: {{pi | number}}
With digitsInfo parameter specified: {{pi | number:'4.1-5'}}
With digitsInfo and locale parameters specified: {{pi | number:'4.1-5':'en'}}

Percentage Pipes

No specified formatting: {{1.2 | percent}}
With digitsInfo parameter specified: {{12.12345 | percent:'4.1-5'}}
